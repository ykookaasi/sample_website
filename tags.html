Important HTMLtags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block-level Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
ol ordered list lists elements in numbered sequence
ul unordered list lists elements
div document division defines division or a section
header section/page header defines header
blockquote long quotation indicate quotation

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>techinical sophistication</em> techinical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt=Michael Hartl">
code code show code <code>def method()</code> def method()
li list item list items <li>item a</li><li>item b</li>
  • item a
  • item b
    		•
    span span mark up part of text <span style="color: blue;">blue</span> eyes. blue eyes.

    Document Tags

    Tag Name Purpose
    <!DOCTYPE html> doctype specifies version of doc HTML
    <html> html represents the root of an HTML document
    <head> head element container for metadata
    <title> title element defines title of the document
    <body> body element defines document body
    <meta> meta element specifies metadata
    <style> style element used to define style info
    <link> link element link to external style sheets
    <script> script element used to define client-side JavaScripts
    <base> base element specifies the base URL/target for all relative URLs
